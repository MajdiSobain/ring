Better RingFastPro
===================

* RingFastPro - Define command (Serial)
* Update RingFastPro - updateList() function - Support (pow)
* Update RingFastPro - updateList() function - Support (Rem) command.
* Update RingFastPro - updateColumn() function - Support (Serial, Pow and Rem)

.. index:: 
	pair: What is new in Ring 1.22?; Better Compiler

Better Compiler
===============

* Ring Compiler - Support (+) operator before numbers.
* Ring Compiler - Lists - Support trailing comma.
* Ring Compiler - Revise operator precedence - Logical AND have higher precedence than Logical OR.
